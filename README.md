# BlackBook - (https://blackbookapplication.herokuapp.com/admin/dashboard)

## Primeiros passos

Clone o repositório
```sh
git clone https://github.com/RafaelMoura29/react-blackbook.git
```

Entre na pasta do projeto
```sh
cd react-blackbook
```

Instale as dependências do projeto
usando npm:
```sh
npm install
```

usando yarn:
```sh
yarn add install
```

Inicie o projeto
```sh
npm start
```
